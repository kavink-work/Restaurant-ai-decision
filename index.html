💬 Prompt history

1 · Problem Framing

Brief: build an AI workflow to find/score/recommend restaurants for a team of 8–12. Chosen area: Siam (Pathum Wan) (later expanded to 5 areas). Decision question + criteria defined.

2 · Data Collection (Apify)

Ran the Apify compass/crawler-google-places actor. Input: searchStringsArray:["restaurant"], locationQuery:"Siam Square, Bangkok", maxCrawledPlacesPerSearch:40, reviews enabled. Exported JSON/CSV. Two sources = Google Maps place data + review data.

3 · Data Storage (Google Sheets)

Apps Script Populate_Restaurant_Sheet.gs built raw_data / clean_data / scoring tabs in one click.

4 · Data Cleaning (Claude)

Prompt gist: "Dedupe, normalise food types, estimate ฿/person where missing, flag low-confidence rows, compute a data-completeness score." Output a clean, comparable table.

5 · Scoring Model (fixed 100 pts)

Prompt gist: "Apply the fixed 6-dimension weights — Rating&Review 25, Group 20, Price 15, Travel 15, Data 15, Unique 10. Rating = 65% stars + 35% review-volume credibility (log). Group from concept type. Price peaks ฿150–350. Document the method." Produced ranked scores + Top 3.

6 · AI Analysis

Prompt gist: "Explain why each Top 3 wins, with trade-offs and cautions, not just praise." Generated reasons + comparison + human-review flags.

7 · Automation (Make.com)

Built a Make scenario: Google Sheets Search Rows (scoring) → Update a Cell (Sheet1!A1 with the live Top pick + timestamp). Ran successfully; set to On-demand to conserve operations.

8 · Enrichment (Google Places API)

Apps Script Enrich_Google_Places.gs / Enrich_Areas.gs pulled real opening hours, price tier, and photo URLs by place_id for all 5 areas — powering hover photos + live Open/Closed badges.

9 · HTML Output

Single-file responsive report: header, objective, workflow diagram, tools, 2+ data sources, scoring table, Top 10, Top 3 (scenario re-ranker), comparison, filterable 40-row dataset with drag-drop CSV re-scoring, area map, human-review queue, EN/TH + light/dark, 5-area switcher.

How each is derived: Rating blends star score (65%) with review-volume credibility (35%, log scale). Group uses concept type — shared / large-table formats (buffet, shabu, Korean BBQ, Thai sharing) score highest. Price peaks at ฿150–350/person (after-work team band). Travel is high for all (Siam Square, 3–6 min from BTS Siam). Data Completeness counts present fields. Uniqueness rewards distinctive concepts (omakase, halal-certified, famous chains).
